SyntaxError: f-string: expecting '}'

Use single quotes inside double-quoted f-strings:

f"Hello, {'John'}"

Or use double quotes inside single-quoted f-strings:

f'Hello, {"John"}'

Python gets confused when it sees the same quote type inside and outside — it can’t tell where the string ends.
Use single quotes inside double-quoted f-strings:

Rule applies even in {}, ex:
f"You've ordered pizza {self.name} prepared with {", ".join(f'{i}: {q}' for i, q in self.ingredients.items())} and the price will be {self.price}lv."
(this will raise the error still)

string formatting for numbers

:.2f
________________________________
    n = 35.75432345
    formatted_number = "%.2f" % n
    print(formatted_number)
    # Output: '35.75'
    # it is a str now

    price = 3.14159
    print(f"Price: {price:.2f}")
    # Output: Price: 3.14
    # all is 1 str


:02d
________________________________
    class Time:
        def __init__(self, hours):
            self.hours = hours

        def formatted_hours(self):
            return f"{self.hours:02d}"

    t = Time(5)
    print(t.formatted_hours())
    # Output: 05


:02d → the format specification:
0 → pad with zeros
2 → width = 2 characters
d → integer (decimal)

Example:
    hours = 5
    print(f"{hours:02d}")
    # Output: 05

    hours = 12
    print(f"{hours:02d}")
    # Output: 12

So :02d means:
print as a 2-digit integer, left-padded with zeros if necessary.

# the result is always a string, no matter what type self.hours is.

unpacking operator *

The * in print(*fruits, sep=', ') is the unpacking operator.

Without *:
fruits = ['apple', 'banana', 'cherry']
print(fruits)
# Output: ['apple', 'banana', 'cherry']
# It prints the list as a list, with brackets and quotes.

With *:
print(fruits)
# Output: apple banana cherry

# The * unpacks the list, so print receives the items as if you had written:
print('apple', 'banana', 'cherry')

Sets

{} → empty dictionary, not set.
set() → empty set.
{1,} → a set with one element (the comma is optional but allowed for single-element sets).

1. Union (| or .union())
    A = {1, 2, 3}
    B = {3, 4, 5}

    A | B # {1, 2, 3, 4, 5}
    A.union(B) # {1, 2, 3, 4, 5}

2. Intersection (& or .intersection())
    A = {1, 2, 3}
    B = {3, 4, 5}

    A & B # {3}
    A.intersection(B) # {3}

3. Difference (- or .difference())
    A = {1, 2, 3}
    B = {3, 4, 5}

    A - B # {1, 2}
    B - A # {4, 5}
    A.difference(B) # {1, 2}

4. Symmetric Difference (^ or .symmetric_difference())
    A = {1, 2, 3}
    B = {3, 4, 5}

    A ^ B # {1, 2, 4, 5}
    A.symmetric_difference(B) # {1, 2, 4, 5}

5. Subset (<=, <) & Superset (>=, >)
    A = {1, 2}
    B = {1, 2, 3}

    A <= B # True (A is a subset of B)
    B >= A # True (B is a superset of A)

    print(A < B) # True
    print(B > A) # True

    A.issubset(B) # True
    B.issuperset(A) # True

>> Reference object:

### shallow copy
    list_a = [1, 2, 3]
    list_b = list_a[:]
    list_b[0] = 5
    print(list_a) # Output: [1, 2, 3]
`list_b = list_a[:]` creates a shallow copy of `list_a`. `list_b` is now a new object, so modifying it does not affect the original `list_a`.


### reference
    list_b = list_a
    list_b[0] = 5
    print(list_a) # Output: [5, 2, 3]
list_b would be just another reference to the same list object. Modifying list_b would also change list_a


### shallow copy - nested
    list_a = [[1, 2], [3, 4]]
    list_b = list_a[:] # shallow copy
    list_b[0][0] = 9
    print(list_a) # Output: [[9, 2], [3, 4]]
Shallow copies only copy the top-level list. If the list contains nested lists or mutable objects, those inner objects are shared. So, changes to nested objects will affect the original, even with a shallow copy.

.split()

.split() is a string method.

It splits a string into a list of substrings based on a separator.

By default, the separator is any whitespace (spaces, tabs, newlines).

    text = "Hello world Python"
    words = text.split()
    print(words)
    # Output: ['Hello', 'world', 'Python']


Using a custom separator
You can pass a string as the separator:

    text = "apple,banana,cherry"
    fruits = text.split(",")
    print(fruits)
    # Output: ['apple', 'banana', 'cherry']


Limiting the number of splits
You can also limit how many splits are made using the maxsplit argument:
    text = "one two three four"
    print(text.split(maxsplit=2))
    # Output: ['one', 'two', 'three four']

.remove(), .pop(), .popitem()

>>> .pop()

Works on lists (and also dictionaries, but differently).
Removes and returns an element at a given index. If no index is given, it removes the last element at (index = -1)

lst = [10, 20, 30, 40]

# Pop last element
x = lst.pop()
print(x) # 40
print(lst) # [10, 20, 30]

# Pop element at index 1
y = lst.pop(1)
print(y) # 20
print(lst) # [10, 30]

>>> .pop() with dictionaries:

d = {'a': 1, 'b': 2}
value = d.pop('a')
print(value) # 1
print(d) # {'b': 2}
For dicts, .pop(key) removes the key and returns its value.


>>> .popitem()

Removes the last inserted key-value pair (or first if last=False).
Designed specifically for “pop the last entry” without needing the key:
d = {'a': 1, 'b': 2, 'c': 3}

# Remove the first inserted item
first_item = d.popitem(last=False)
print(first_item) # ('a', 1)
print(d) # {'b': 2, 'c': 3}

# Remove the last inserted item
last_item = d.popitem() # last=True by default
print(last_item) # ('c', 3)
print(d) # {'b': 2}

In Python 3.7+, dictionaries preserve insertion order, so you can pop the “last key” using .popitem(). The reason is that .pop() in a dict requires you to specify a key. Unlike lists, dictionaries aren’t indexed by position, they are keyed collections. You cannot just call .pop() with no arguments like you do for a list, because “the last item” isn’t inherently known to the method—it’s keyed, not indexed.


.remove()
Works only on lists.
Removes the first occurrence of a specific value.
Does not return the value.
lst = [10, 20, 30, 20]
lst.remove(20)
print(lst) # [10, 30, 20]
# Removes the first 20, not all of them.

To remove all occurrences:
    lst = [1, 2, 3, 2, 4, 2]
    while 2 in lst:
        lst.remove(2)
    print(lst) # [1, 3, 4]

    lst = [1, 2, 3, 2, 4, 2]
    lst = [x for x in lst if x != 2]
    print(lst) # [1, 3, 4]
